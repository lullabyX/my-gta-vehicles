# What is it?

Keep track of the vehicles that you own in Grand Theft Auto: Online. Add the vehicles and their storage, look up, sort, see details.

# Features

* Add Vehicles with storage location. Optionally add comment.
* Single-Tap on a vehicle row to view the vehicle details.
* Double-Tap on a vehicle row to edit/delete the vehicle.
* Sort, search on different vehicle attributes

# Want to run and modify the app in your machine?

### You will need Git and NodeJS installed before continuing. 

* Open your favorite shell and clone the repository by running `git clone https://github.com/lullabyX/my-gta-vehicles.git` or download the zip
* Go to the repository that you've just cloned in your shell and run `npm install` to install the necessary dependencies. 
* After that, run `npm start` to run the wep app in your machine. A new tab will open in your browser running the app. (you might need to press `y` after `npm start`)
